**Python project** that manages a movie catalog using dictionaries.

## Overview
This script simulates a simple **movie rating system** where movies can be added, rated, and analyzed.

## What it does
- **Stores movies** with year, rating, and votes
- **Adds new movies** to the catalog
- **Updates ratings** using a weighted average
- **Finds the highest-rated movie**
- **Calculates the average rating** of the catalog

## How it works
- Uses **dictionaries** for data storage
- Applies **basic validation rules**
- Prints all results to the **console**

## Requirements
- **Python 3.x**

## Usage
Run the script:
    python movie_rating_manager.py

## Output
- Console output with:
  - **updated ratings**
  - **best-rated movie**
  - **full catalog**
  - **average rating**

## Notes
Built to practice **dictionaries**, **functions**, and **clean data manipulation**.
